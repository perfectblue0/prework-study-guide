HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title element defines the title of the web page and is important for Search Engine Optimization(SEO)
  • The alt text in image elements are important for accessibility(allow screen readers to share descriptions with users) and to improve SEO
  • The section elements are semantice elements that define a section in a document that contains thematic content

CSS

  • Using external CSS stlesheets are the most preferred way to style a webpage because they allow developers to keep all their CSS rules in a seperate file, which makes designing easier.
  • A selector defines the element/attribute to which the rules or declarations, will apply.
  • Using * as a selector in CSS is a special wild card symbol that allows a rule(s) to be applied to all elements visible on the page.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git reset [commit]: undoes all commits after [commit], preserving files locally
  • git branch -d local_branch_name: deletes a local branch; one must not be on the branch one wants to delete
  • git add "file": adds files you want to to commit to staging
  • git commit -m "description of changes": commits and adds the description of changes you have made in the branch

JavaScript

  • Variable are containers that store values.
  • Control flow is the order in which a computer executes code in a script.
  • Comments are snippets of texts that the browser will ignore.
  • Operators are mathematical symbols that produce a result based on 2 values or variables.
  • Undefined is a data type that indicates the absence of a value.
  • Null is a data type that indicates the absence of an object value.
  • NaN is a special kind of number value that's typically encountered when the result of an arithmetic operation cannot be expressed as a number.
